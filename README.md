#### 常用git指令

```bash
git clone [url]  # 从仓库中克隆到本地
git add .        # 添加全部变动
git commit [-m "message"]  # 提交版本库
git push        # 将本地版本库推至远端服务器

git pull        # 从服务器拉取最新版本
git checkout -b [name of new branch] master   # 从master分支切换到新分区
git branch      # 查看分支
git branch -d [branch]  # 删除分支
git merge [。。。]   # 分支合并
```
